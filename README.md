# Crypto Dashboard

This is crypto dashboard for bali united test interview

## Quick start

1. Create Laravel Projects:
   ```
   laravel new crypto_dashboard
   ```
   - choose no starter kit
   - choose pest testing framework
   - use sqlite database
   - choose no to database migrations

2. Disable the database:
   ```
   SESSION_DRIVER=array
   ```

3. Install dependency:
   ```
   npm install
   ```

4. Copy all the source from github and replace it in your laravel project

5. Run the project
   ```
   composer run dev
   ```
